# 🔐 Password Generator

A simple yet secure Python-based password generator that creates random, strong passwords using a combination of lowercase letters, uppercase letters, digits, and special symbols.

## ✨ Features

- **Customizable Length**: Choose password length between 1-94 characters
- **Character Variety**: Uses 94 different characters including:
  - 26 lowercase letters (a-z)
  - 26 uppercase letters (A-Z)
  - 10 digits (0-9)
  - 32 punctuation symbols
- **No Repetition**: Each character appears only once in the password
- **Input Validation**: Handles invalid inputs and edge cases
